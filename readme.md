## Lixulun's Ninety-Nine Ocaml Problems solution

Ref: https://ocaml.org/exercises

### Examples

```
  $ ocaml test.ml  # run all tests
  $ ocaml test.ml list  # list all exercises
  $ ocaml test.ml 01_xxx_xxx.ml # run specified exercise
  $ ocaml test.ml 01_xxx.ml 02_xxx.ml ... #  run specified exercises
  $ ocaml test.ml help / -h / --help  # print this
```
